# Git installation process

## Step-1

### `sudo apt install git` [For ubuntu]

## Step-3
### `git --version`

## Step-4
### `git config --global user.name "dev"`
### `git config --global user.email "dev@gmail.com"`
### `nano ~/.gitconfig`

## Step-5
### `ssh-keygen`
Press enter key 2 times

## Step-6
### `cat ~/.ssh/id_rsa.pub`

Copy the key and add git account as ssh key
